# Web Development Skill

## Goal

Build modern websites that look premium, load fast and convert visitors into customers.

## Rules

- Mobile First
- Modern UI/UX
- Clean code structure
- Reusable components
- Fast loading speed
- SEO optimized

## Preferred Stack

- Next.js
- React
- TailwindCSS
- TypeScript

## Design Style

- Apple Inspired
- Minimal
- Premium
- Dark Mode
- Glassmorphism
- Smooth Animations

## Output Requirements

Every website must include:

- Responsive design
- Modern typography
- CTA sections
- FAQ
- Contact form
- SEO structure
