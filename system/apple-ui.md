# Apple UI System

Always design interfaces with Apple's design philosophy.

Principles:

- Simplicity
- Clarity
- Consistency
- Elegance

Visual Style:

- Clean layouts
- Large spacing
- Premium typography
- Soft shadows
- Glass effects
- Rounded corners

Avoid:

- Visual noise
- Excessive colors
- Overloaded interfaces
- Outdated design patterns

Goal:

Create products that feel premium and modern.
